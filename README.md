# Hormozi Skills for Claude Code

Claude Code skills based on the Alex Hormozi trilogy:
- [$100M Offers](https://www.acquisition.com/books)
- [$100M Leads](https://www.acquisition.com/books)
- [$100M Money Models: How To Make Money](https://www.acquisition.com/books)

## Install

In Claude Code:

```
/plugin marketplace add zacker-tech/hormozi-skills
/plugin install hormozi
```

Or clone locally:

```
git clone https://github.com/zacker-tech/hormozi-skills.git ~/.claude/plugins/hormozi-skills
```

Then in Claude Code:

```
/plugin install ~/.claude/plugins/hormozi-skills
```

## Skills

| Skill | Command | When to use |
|-------|---------|-------------|
| Grand Slam Offer | `/grand-slam-offer` | Creating an irresistible offer from scratch |
| Value Equation | `/value-equation` | Pricing your product or service |
| Niche Down | `/niche-down` | Choosing who to sell to |
| Dream Outcome | `/dream-outcome` | Writing copy, sales pages, or pitches |
| Lead Magnet | `/lead-magnet` | Building a free offer to attract leads |
| Warm Outreach | `/warm-outreach` | Getting first customers from your network |
| Content Engine | `/content-engine` | Building an audience-driven lead machine |
| Core Four | `/core-four` | Choosing your primary customer acquisition method |
| Money Model | `/money-model` | Structuring your revenue model for max profit |
| Hormozi Review | `/hormozi-review` | Gut-checking any business decision against Hormozi principles |

The skills follow the trilogy's progression:

- **Offer** — Start with an offer so good people feel stupid saying no
- **Leads** — Build systems to get strangers to want to buy
- **Money** — Structure your business to keep what you make
